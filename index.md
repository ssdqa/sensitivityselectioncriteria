# Sensitivity to Selection Criteria

The Sensitivity to Selection Criteria module summarizes demographic
makeup and fact distribution differences between a base cohort
definition and alternate cohort definitions. This functionality will
allow users to assess which cohort definition best fits their needs in a
study context.

## Installation

You can install the development version of this package like so:

``` r
devtools::install_github('ssdqa/sensitivityselectioncriteria')
```

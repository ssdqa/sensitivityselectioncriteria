# *Single Site, Anomaly Detection, Cross-Sectional*

*Single Site, Anomaly Detection, Cross-Sectional*

## Usage

``` r
ssc_ss_anom_cs(process_output)
```

## Arguments

- process_output:

  the output from ssc_process

## Value

a heat map displaying the standardized mean difference between the base
cohort and each of the alternate cohorts
